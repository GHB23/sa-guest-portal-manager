Google Sheets Integration

Google Sheets as Database

Instead of Airtable and Glide, we use Google Sheets which is:

  • Free to use
  • Familiar to most users
  • Accessible from any device
  • Easy to share with team members
How it works:
  1. Create a Google Sheet with your property data
  2. Use Google Apps Script to create a web API
  3. Connect your guest portal to the API
  4. Data updates in real-time
View Template Sheet

Sample Data Structure

Your Google Sheet should have these columns:

Column Name Description
Property NameName of the apartment/guest house
AddressFull physical address
WhatsApp NumberContact number for verification
Check-in DateGuest arrival date
Check-out DateGuest departure date
WiFi NetworkNetwork name for the property
WiFi PasswordNetwork password
Owner EmailProperty owner's email for reports
StatusActive/Inactive

This structure allows you to manage multiple properties efficiently.

Implementation Steps

1. Create Your Google Sheet

Set up a new Google Sheet with the column structure shown above. Add your property data.

2. Set Up Google Apps Script

Create a script to expose your sheet data as a web API that your portal can access.

3. Integrate with Your Portal

Connect your guest portal to the API to fetch and update property information.

4. Test the System

Verify that QR codes work correctly and data is being retrieved properly.
